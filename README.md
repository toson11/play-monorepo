# Monorepo 架构
Monorepo（单一代码库）是一种将多个项目存放在同一个代码库中的软件开发方法。

## Monorepo 与 Turborepo
monorepo 是一种“管理多个项目的方式”
Turborepo 是“帮助你高效管理和构建 monorepo 的工具”

## Turborepo 的优势
- **远程缓存**：会缓存构建结果，避免重复构建，提高构建速度。
- **并行处理**：自动优化任务调度，最大化并行处理，充分利用CPU资源。
- **逐步引入**：几分钟内即可将现有项目迁移到 Turborepo，无需大规模重构。
